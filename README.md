# Promake AI Case

Bu proje bir şablon oluşturucu uygulaması. Sayfa bölümlerini (Header, Hero, Features) seçip birleştirerek şablonlar oluşturabilirsin.

## Nasıl Çalışır?

Her bölümün farklı varyasyonları var ve bunlar `src/sections` klasöründe tutuluyor. Her varyasyonun kendine özel `id` ve `description`'u var. İstenilen componentler seçilerek bir sayfa render alınır.

## Teknolojiler

- React + TypeScript
- TailwindCSS ve ShadCN UI bileşenleri
- Context API ile state yönetimi

## Kurulum

```bash
npm install
npm run dev
```
